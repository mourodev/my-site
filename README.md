# Meu Portfólio

Um espaço simples e leve para apresentar meu trabalho como desenvolvedor frontend.

## O que é

Este é o meu site pessoal. A ideia é ter um lugar limpo e direto para mostrar projetos, experiência e formas de contato — sem complicação.

## Tecnologias

- **Astro** — framework moderno que entrega sites rápidos e leves
- **Sharp** — otimização de imagens
- **Astro Icon** — ícones de forma prática

Para manter a qualidade do código, uso o **Ultracite** com Biome — tudo formatado e padronizado automaticamente.

## Scripts

```bash
pnpm dev      # inicia o servidor local
pnpm build    # gera a versão de produção
pnpm preview  # visualiza a build antes de publicar
pnpm fix      # formata e corrige o código
```
